### 2. 핵심 정리 (내 언어로)
- Scanner sc = new Scanner(System.in); 로 입력을 받는다. import java.util.Scanner;가 필요하고, 사용이 끝나면 sc.close();로 닫는다.
- 정수 입력은 sc.nextInt()를, 문자열 입력은 sc.nextLine()을 사용한다.
- 배열 생성- String[] members = new String[num];로 num 크기 만큼 문자열을 저장할 수 있는 배열(members)을 만든다.
- sc.nextLine(); - sc.nextInt() 이후 입력 버퍼에 남아있는 개행 문자(\n)를 제거하여, 이후 nextLine()으로 정상적인 문자열 입력을 받기 위해 사용한다.
- while문과 for문 비교 - while문은 반복 횟수가 정해지지 않았을 때, for문은 반복횟수가 정해져 있을 때 주로 사용한다.
- print, println, printf의 차이 - print()는 출력 후 줄바꿈 없고, println()은 출력 후 자동 줄바꿈하고, printf()는 형식 지정 출력(format 출력)
